# 🔐 Monoalphabetic Cipher

A desktop encryption tool built with Python and Tkinter
that encrypts and decrypts text using a substitution cipher.

## ✨ Features
- Encrypt & Decrypt text instantly
- Generate random substitution keys
- View the cipher key mapping
- Copy result to clipboard
- Simple GUI built with Tkinter

## 🛠️ Tech Stack
Python · Tkinter · Cryptography

## 🚀 How to Run
python monoalphabetic_cipher.py

## 📸 How It Works
- Each letter maps to another letter (A→Q, B→W...)
- Random key generator creates new mappings
- Reverse key used for decryption
